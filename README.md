# tormaroe.github.io

This repository contains the source code for my personal landing page and portfolio website, hosted at [https://tormaroe.github.io](https://tormaroe.github.io).

## 🛠️ Local Development

You can run the site locally to preview any changes:

1. **Install dependencies** (required if you are using image optimization tools like `sharp`):
   ```bash
   npm install
   ```

2. **Start the local server**:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.
